[ ПЕРВИЧНАЯ НАСТРОЙКА ]

1. Скопировать файл appsettings.example.json в appsettings.json

   Можно его скопировать в тот же самый каталог,
   а можно в каталог "C:\Users\<USERNAME>\AppData\Roaming\NZ.RdpMaid"

   При старте программа попытается прочитать файл в текущем каталоге,
   если его там нет, то будет искать в "C:\Users\<USERNAME>\AppData\Roaming\NZ.RdpMaid".

   Если нет ни там ни сям, то программа не сможет запуститься и упадёт.

2. Открыть файл appsettings.json
3. В свойстве "Login" заменить значение "LOGIN" на ваш логин
4. В свойстве "Address" заменить значение "ADDRESS" на имя удалённого компьютера

[ ОБНОВЛЕНИЕ ]

1. Скопировать все файлы из пакета обновления в каталог с программой с заменой существующих файлов
2. Открыть файлы appsettings.example.json и appsettings.json
3. Скопировать недостающие настройки из appsettings.example.json в appsettings.json

[ ПРИМЕЧАНИЕ 1 - РЕДАКТИРОВАНИЕ RDP-ФАЙЛА ]

Начиная с версии 0.7.0 в каталоге установки программы расположен шаблон RDP-файла "<ПУТЬ_УСТАНОВКИ>\Resources\Sessions\TemplateSession.rdp".
При первом подключении файл будет скопирован в каталог пользовательских настроек "C:\Users\<USERNAME>\AppData\Roaming\NZ.RdpMaid\TemplateSession.rdp".
Файл можно отредактировать, чтобы поменять настройки RDP-сессии.
При обновлении программы файл в каталоге установки будет перетёрт и изменения потеряются, однако его копия в каталоге пользовательских настроек останется.
При подключении приоритет отдаётся файлу в каталоге пользовательских настроек.
